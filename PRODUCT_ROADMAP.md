# Анализ развития продукта "Фото для резюме"

**Дата:** 2026-01-16

---

## Текущее состояние

MVP работает: пользователь выбирает пол → отправляет фото → получает студийный портрет за ~28 сек.

---

## Направления развития

### 1. Расширение сценариев использования

**Выбор типа фото (ключевая фича):**
- **Резюме/LinkedIn** — строгий деловой стиль, нейтральный фон
- **Корпоративный портал** — полуформальный, с лёгкой улыбкой
- **Социальные сети** — более живой, creative casual
- **Спикер/эксперт** — уверенная поза, динамичное освещение

**Выбор дресс-кода:**
- Строгий деловой (костюм/пиджак)
- Business casual (рубашка/блузка)
- Creative (для IT, дизайнеров)
- Медицина (халат)
- Юридические услуги

### 2. Персонализация результата

**Настройки перед генерацией:**
- Цвет фона (белый, серый, градиент, корпоративный цвет)
- Кадрирование (только лицо, по грудь, по пояс)
- Настроение (нейтральное, дружелюбное, уверенное)
- Ретушь (минимальная, умеренная, полная)

**Пакетная генерация:**
- Генерация 3-5 вариантов за один запрос
- Пользователь выбирает лучший

### 3. Качество и форматы

**Выходные форматы:**
- Квадрат 1:1 (для аватаров)
- 3:4 (для резюме)
- 16:9 (для обложек LinkedIn)
- Оригинальное разрешение

**Улучшение качества:**
- Апскейл до 2K/4K
- Удаление артефактов
- Профессиональная цветокоррекция

### 4. Монетизация

**Freemium модель:**
```
Бесплатно: 1-2 генерации с водяным знаком
Базовый (299₽): 10 генераций без знака
Про (699₽): безлимит + все стили + высокое разрешение
Бизнес (от 2999₽): API для HR-платформ
```

**Дополнительные услуги:**
- Экспресс-генерация (приоритетная очередь)
- Ручная ретушь от дизайнера
- Пакет "Полный профиль" (фото + баннер + визитка)

### 5. Обратная связь и улучшение качества

**Оценка генерации:**
- После получения результата — запрос оценки от 1 до 5 звёзд
- При оценке ниже 4 — дополнительный вопрос: «Чего не хватило до 5?»
  - Варианты: качество лица, освещение, одежда, фон, поза, другое (свой текст)
- Сбор статистики для улучшения промптов и модели
- Возможность переделать с учётом замечаний

### 6. Удержание пользователей

**История и портфолио:**
- Сохранение всех генераций в профиле
- Возможность скачать повторно
- "Мои стили" — сохранённые настройки

**Регулярное использование:**
- Напоминание обновить фото (раз в год)
- Сезонные стили (новогодний, летний)
- Интеграция с календарём (фото к конференции)

### 7. B2B направление

**HR-интеграции:**
- API для рекрутинговых платформ
- Массовая обработка для компаний
- White-label решение

**Корпоративные клиенты:**
- Единый стиль для всей команды
- Брендированные фоны с логотипом
- Оптовые тарифы

### 8. Техническое развитие

**Приоритетные задачи:**
- Webhook вместо polling (надёжность)
- База данных для истории (PostgreSQL)
- Очередь задач (Redis/Celery)
- Кэширование промптов

**UX улучшения:**
- Предпросмотр стиля (примеры до генерации)
- Прогресс-бар генерации
- Сравнение "до/после"
