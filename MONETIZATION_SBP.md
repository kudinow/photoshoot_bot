# Монетизация Photoshoot AI Bot — Полный обзор

## Варианты монетизации для России

### 1. Telegram Stars (Рекомендуется)

Встроенная валюта Telegram — работает без внешних платёжных систем:
- Пользователи покупают Stars в App Store/Google Play
- Бот принимает Stars через Telegram Payments API
- Вывод: Telegram конвертирует в TON или фиат

**Плюсы:** Нет блокировок, встроено в Telegram, простая интеграция через aiogram

---

### 2. ЮKassa (ex-Яндекс.Касса)

Полноценный эквайринг для РФ:
- Карты Мир, Visa, Mastercard (российские)
- СБП (оплата по QR)
- ЮMoney кошелёк

**Плюсы:** Надёжно, привычно для пользователей
**Минусы:** Нужно ИП/ООО, комиссия ~3%

---

### 3. СБП напрямую

Оплата по QR-коду или номеру телефона:
- Можно через Тинькофф, Сбер API
- Или просто статический QR + ручная проверка

---

### 4. Криптовалюта (TON)

TON Connect + TON кошелёк:
- Популярно в Telegram-экосистеме
- Без посредников

**Минусы:** Не все пользователи готовы к крипте

---

## СБП — Детальный разбор

### Простой вариант (без API)

#### Схема работы

1. Пользователь нажимает «Оплатить»
2. Бот отправляет QR-код или ссылку на оплату (статический QR привязан к вашему счёту)
3. Пользователь платит через приложение банка
4. Вы вручную или полуавтоматически подтверждаете оплату

#### Проблема сопоставления платежей

Нужно понять, какой платёж от какого пользователя. Решения:

**Вариант А — Комментарий к платежу:**
- Просить указать в комментарии свой Telegram ID или код
- Минус: пользователи забывают/ошибаются

**Вариант Б — Уникальная сумма:**
- Генерировать уникальную сумму для каждого пользователя (149.37₽, 149.42₽, 149.58₽)
- Бот запоминает: user_id → сумма
- При получении платежа сопоставляет по сумме
- Плюс: автоматизируемо

---

### Продвинутый вариант (с API банка)

#### Схема работы

```
Пользователь → «Купить 5 генераций»
     ↓
Бот создаёт платёж через API банка
     ↓
Банк возвращает динамический QR/ссылку
     ↓
Пользователь оплачивает
     ↓
Банк отправляет webhook боту
     ↓
Бот автоматически начисляет генерации
```

#### Требования для API

- ИП или ООО (самозанятым недоступно)
- Расчётный счёт в банке с поддержкой СБП API
- Интеграция с SDK банка

#### Банки с СБП API

| Банк | Документация | Комиссия |
|------|--------------|----------|
| Тинькофф | [developers.tinkoff.ru](https://developers.tinkoff.ru) | ~0.4-0.7% |
| Сбер | [developers.sber.ru](https://developers.sber.ru) | ~0.4-0.7% |
| Альфа | [developers.alfabank.ru](https://developers.alfabank.ru) | ~0.4-0.7% |

## Сравнение вариантов СБП

| Аспект | Статический QR | API банка |
|--------|----------------|-----------|
| Сложность | Минимальная | Средняя |
| Автоматизация | Ручная/частичная | Полная |
| Требования | Любой счёт | ИП/ООО + договор |
| Комиссия | 0% (p2p) или ~0.4% | ~0.4-0.7% |
| Масштаб | До ~50 платежей/день | Без ограничений |

---

## Сравнение всех способов оплаты

| Способ | Сложность | Автоматизация | Требования | Комиссия |
|--------|-----------|---------------|------------|----------|
| Telegram Stars | Низкая | Полная | Ничего | ~30% (Apple/Google) |
| ЮKassa | Средняя | Полная | ИП/ООО | ~3% |
| СБП (статический QR) | Минимальная | Частичная | Любой счёт | 0% |
| СБП (API) | Средняя | Полная | ИП/ООО | ~0.4-0.7% |
| TON | Средняя | Полная | Ничего | ~0% |

## Рекомендация по выбору

1. **Для быстрого старта** — Telegram Stars (простая интеграция, работает из коробки)
2. **Для тестирования гипотезы** — статический QR СБП с уникальными суммами
3. **Для серьёзного бизнеса** — ЮKassa + СБП (полная автоматизация, все способы оплаты)
